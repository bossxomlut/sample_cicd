# Rust REST API Example

This project is a simple REST API built with Rust and Actix-web.

## Features
- GET /hello: Returns "Hello"

## Usage
1. Build the project:
   ```sh
   cargo build
   ```
2. Run the server:
   ```sh
   cargo run
   ```
3. Access the endpoint:
   - Open your browser or use curl:
     ```sh
     curl http://127.0.0.1:8080/hello
     ```

## Requirements
- Rust (edition 2021)
- actix-web
